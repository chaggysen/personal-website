In this project, multiple software engineering best practices are being implemented, such as the creation of a domain model that shows all the attributes, concepts and relationships in a UML class diagram. and designed a fully functional state machine diagram using Yakindu state chart simulation software, unit testing by developing controller methods and mapping of Gherkin (Cucumber) scenarios and ensure that all Gherkin scenarios initiated by Cucumber passed successfully as acceptance tests and the implementation of MVC design architecture. The technologies used are : Java, JUnit, Java Swing, Java 2D, Cucumber, Umple, Yakindu State Chart.
